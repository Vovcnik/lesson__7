Терминатор в backgroundЕ

Универсальное свойство background позволяет установить одновременно несколько характеристик фона. Значения могут идти в любом порядке, браузер сам определит, какое из них соответствует нужному свойству. Для подробного ознакомления смотрите информацию о каждом свойстве отдельно. Допустимо, также, указывать параметры сразу нескольких фонов, перечисляя их через запятую.

  • background-color
  • image
  • Background-position
  • Background-size
  • background-attachment
  • background-attachment

Запись

background-color: orange; background-image: url("../img/image.jpg"); background-position: center; background-repeat: no-repeat; repeat-x | repeat-y | [repeat | space | round | no-repeat]{1,2} background-size: cover; [ <размер> | <проценты> | auto ]{1,2} | cover | contain background-attachment: fixed; fixed | scroll | local[, fixed | scroll | local]*
или background: orange url("../img/image.jpg") center / cover no-repeat fixed;

background-color

background-color: #fff;

radial-gradient

Функция radial-gradient() добавляет радиальный градиент к фону элемента. Радиальные по своему принципу похожи на линейные градиенты, но один цвет переходит в другой не вдоль прямой линии, а словно круги по воде вокруг точки.
background: -webkit-radial-gradient(center, ellipse cover, #1e5799 0%, #2989d8 50%, #207cca 51%, #7db9e8 100%);

Background-image

Устанавливает фоновое изображение для элемента. Если одновременно для элемента задан цвет фона, он будет показан, пока фоновая картинка не загрузится полностью. То же произойдет, если изображения не доступны или их показ в браузере отключен. В случае наличия в рисунке прозрачных областей, через них будет проглядывать фоновый цвет. Допустимо указывать несколько фоновых изображений, перечисляя их параметры через запятую. background-image: url(../img/31520.jpg);

Background-repeat

Определяет, как будет повторяться фоновое изображение, установленное с помощью свойства background-image. Можно установить повторение рисунка только по горизонтали, по вертикали или в обе стороны. Допустимо указывать несколько значений для каждого фона, перечисляя значения через запятую.
Значение по умолчанию repeat background: url(../img/31520.jpg) no-repeat;

repeat-x

background: url(../img/31520.jpg) center /150px repeat-x;

repeat-y

background: url(../img/31520.jpg) 0 0 /150px repeat-y;

space

Изображение повторяется столько раз, чтобы полностью заполнить область; если это не удаётся, между картинками добавляется пустое пространство.
background: url(../img/31520.jpg) 0 0 /100px space;

Background-position

Задаёт начальное положение фонового изображения, установленного с помощью свойства background-image. Допустимо указывать несколько значений для каждого фона, перечисляя значения через запятую.
Значение по умолчанию 0% 0%
background: url(../img/31520.jpg) 0 0 /100px no-repeat;

Background-position-center

background: url(../img/31520.jpg) center /200px no-repeat; или
center = 50% 0% (по центру вверху)

Background-position-left

background: url(../img/31520.jpg) left /200px no-repeat; или
left = 0% 50% (по левому краю и по центру)

Background-position-right

background: url(../img/31520.jpg) right /200px no-repeat;
right = 100% 0% (в правом верхнем углу)

Background-position-bottom

background: #fff url(../img/31520.jpg) bottom /200px no-repeat; или
bottom center = bottom = 50% 100% (по центру внизу)

Background-size:

<размер> Задаёт размер в любых доступных для CSS единицах — пикселях (px), сантиметрах (cm), em и др. <проценты> Задаёт размер фоновой картинки в процентах от ширины или высоты элемента. auto Если задано одновременно для ширины и высоты (auto auto), размеры фона остаются исходными; если только для одной стороны картинки (100px auto), то размер вычисляется автоматически исходя из пропорций картинки.
background: url(../img/31520.jpg) 0 0 /200px no-repeat;

Background-size:cover

background: url(../img/31520.jpg) center /cover no-repeat;

Background-size:contain

Масштабирует изображение с сохранением пропорций таким образом, чтобы картинка целиком поместилась внутрь блока. Если установлено одно значение, оно устанавливает ширину фона, второе значение принимается за auto. Пропорции картинки при этом сохраняются. Использование двух значений через пробел задаёт ширину и высоту фоновой картинки.
background: url(../img/31520.jpg) center /contain no-repeat;

background-attachment:fixed

Делает фоновое изображение элемента неподвижным при скроле background: url(../img/31520.jpg) center /contain,fixed no-repeat;

background-attachment:scroll

scroll Позволяет перемещаться фону вместе с содержимым.
background: url(../img/31520.jpg) center /800px scroll no-repeat;

background-attachment:local
